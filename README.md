# A Cup of Coffee 개인 프로젝트
----------------------------------------

## Main



- 로고 클릭식 메인 화면 출력

- user 아이콘 클릭시 user 페이지 이동

- 검색창 아이콘/검색창 입력시 map 페이지로 이동

- 검색창 인풋 내용 로컬스토리지 저장 -> map 페이지 인풋에 출력

----------------------------------------

## Map



- 구글맵 API

- 첫페이지 현재위치 출력

- 위치 확인 불가시 신촌역 기본 출력

- 로고 클릭시 메인화면 이동

- user 아이콘 클릭시 user 페이지 이동

----------------------------------------

## User (My-List)



- 타이틀 클릭시 메인 페이지 이동

- FAQ 클릭시 모달 오픈(리액트 구현)

    모달 메뉴 아코디언

    백그라운드 스크롤방지
